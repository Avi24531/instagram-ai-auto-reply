# instagram-ai-auto-reply

Instagram AI Auto Reply Automation
Overview

Automatically responds to Instagram DMs using AI and stores leads in Google Sheets.

## Features
-Instagram webhook integration
-AI-generated replies
-FAQ retrieval
-Lead capture
-Google Sheets CRM
-Tech Stack
-n8n
-OpenAI
-Google Sheets
-Meta Graph API

## Workflow

Instagram DM
    ↓
Webhook
    ↓
AI Agent
    ↓
FAQ Search
    ↓
Lead Extraction
    ↓
Google Sheets
    ↓
Instagram Reply

## Setup

Import workflow.json
Connect Meta credentials
Connect OpenAI API
Connect Google Sheets
Activate workflow
